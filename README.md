# UCW BedChamp

Custom scoreboard and move engine for the Ultimate Couple Wrestling BedChamp experience.

## Features
- Multiple match types (Quick, Best-of-3, Endurance, Iron Woman, Sudden Death, Practice, Erotic Fight)
- Toggleable move deck with artwork-driven descriptions
- Sudden-Death trophy prompts and coin-flip opener
- Erotic fight overlay with timer, orgasm tracking, and big tap targets

## Getting Started
```bash
npm install
npm run dev
```
The dev server runs on the port selected by Vite (default: http://localhost:5173).

## Build
```bash
npm run build
```
Outputs static assets in `dist/` ready for deployment.

## Lint
```bash
npm run lint
```
Runs ESLint using the project configuration.
